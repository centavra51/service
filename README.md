# Service

Статический лендинг с формой заявки и серверной функцией для отправки лидов в Telegram через Vercel.

## Что нужно для деплоя

- GitHub-репозиторий
- Проект в Vercel, подключенный к этому репозиторию
- Переменные окружения в Vercel:
  - `TELEGRAM_BOT_TOKEN`
  - `TELEGRAM_CHAT_ID`

## Как это работает

- `index.html`, `style.css`, `script.js` отдают статическую страницу
- `submit-api.js` обрабатывает `POST /api/submit`
- `vercel.json` маршрутизирует запросы на серверную функцию и SPA-страницу

## Проверка после деплоя

1. Открыть сайт и переключить язык `RU/RO`
2. Нажать `Рассчитать`, проверить обновление суммы
3. Отправить тестовую заявку
4. Убедиться, что сообщение пришло в Telegram
